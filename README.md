# Pixiv Bookmark Roulette
A user script that selects and opens one random illustration from your pixiv bookmarks.

## Usage
1. Navigate to any page on [pixiv](https://www.pixiv.net/).
2. Ensure you are logged in to your account.
3. Click the extension icon in your browser's toolbar to open the user script manager menu.
4. Select "Open Random Bookmark"
5. The script will calculate a random target from both your public and private bookmarks and redirect you to the artwork's page.

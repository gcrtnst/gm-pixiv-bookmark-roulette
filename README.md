# Pixiv Random Bookmark

A lightweight userscript that adds a menu command to instantly jump to a random illustration from your public and private pixiv bookmarks.

## Usage

1. Navigate to any page on [pixiv](https://www.pixiv.net/).
2. Ensure you are logged in to your account.
3. Open your userscript manager's menu (usually by clicking the extension icon in your browser toolbar).
4. Select "Open Random Bookmark".
5. The script will calculate a random target and redirect you to the artwork's page.

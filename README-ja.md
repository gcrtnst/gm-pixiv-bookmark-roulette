# Pixiv ブックマークガチャ
pixiv ブックマークからランダムなイラストを1つ選んで開くユーザースクリプトです。

## 使い方
1. [pixiv](https://www.pixiv.net/) 上の任意のページを開く。
2. pixiv にログインしていることを確認する。
3. ブラウザのツールバーにある拡張機能アイコンをクリックして、ユーザースクリプトマネージャーのメニューを開く。
4. 「ブックマークからランダムに開く」を選択。
5. スクリプトが 公開/非公開 双方のブックマークからランダムな対象を1つ選択し、その作品ページにリダイレクトします。
